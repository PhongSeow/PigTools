# PigTools
豚豚工具集是一个简化一些 .net 开发接口的工具集类库。

## ***目录和文件描述***

### Release

执行码目录，如果你不想看到源程序，你可以直接使用这个目录中的文件。

##### Release\DotNet\PigToolsLib
PigToolsLib 的类库 DLL 和 NuGet 包。

##### Release\DotNet\PigToolsWinLib
PigToolsWinLib 的类库 DLL 和 NuGet 包。

##### Release\GetFileAndDirList\VB6
获取文件和目录列表的工具执行码，免安装，可以直接在不同版本的Windows上运行。
执行 GetFileAndDirList.exe ，会在当前目录下生成 DirList.txt 和 FileList.txt 。

### Src

源码目录。

#### Src\DotNet\PigToolsLib

PigToolsLib 源码目录

#### Src\DotNet\PigToolsWinLib

PigToolsWinLib 源码目录

#### Src\DotNet\PigToolsDemo

PigToolsLib 示例目录。

#### Src\VB6\GetFileAndDirList

获取文件和目录列表的工具源码目录
